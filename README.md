# Surnames First

Created by **Roy Galvin**.

Reformat name lists for CIE remembrance mass: **surname first**, then first name(s), with any birth/death or “Died” dates unchanged at the end.

**Live page:** [https://gally74.github.io/surnames-first/](https://gally74.github.io/surnames-first/)

## How to use

1. In your Word document, copy the list of names (one name per line).
2. Open the [Surnames First](https://gally74.github.io/surnames-first/) page.
3. Paste the names into the input box.
4. Click **Reformat**.
5. Click **Copy result**, then paste back into your Word document.

## Examples

| You paste | You get |
|-----------|--------|
| Joe Bloggs 1999 - 2008 | Bloggs, Joe 1999 - 2008 |


Supports nicknames in parentheses, titles (Fr., Mr., Mrs.), compound surnames (e.g. O'Shea, St. Devlin, Gray Snr.), and both year ranges (1927 - 2008) and “Died DD/MM/YYYY” at the end.
